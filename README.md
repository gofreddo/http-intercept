# http-intercept

> Intercepts node http.request. Allows conditional request modification.

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i http-intercept --save-dev
```

## Usage

```js
var http-intercept = require('http-intercept');
```

## Running tests

Install dev dependencies:

```sh
$ npm i -d && npm test
```
